Proyecto: Plataforma Creativa

Descripción del Proyecto

Este proyecto busca crear una plataforma digital donde artistas, fotógrafos y diseñadores puedan compartir su trabajo, conectarse y colaborar en tiempo real. Su objetivo es fomentar la comunidad creativa y promover el talento independiente.

Tecnologías Utilizadas

  • HTML
  • CSS
  • JavaScript
  • Python (para backend)
